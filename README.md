# Turnstile

Turnstile is a Proof Assistant based on **System LK** (Classical Sequent Calculus), implemented in modern C++.

## Project Structure

- `src/`: Source files (.cpp)
- `include/`: Header files (.hpp)
- `tests/`: Unit tests
- `build/`: Build directory (generated by CMake)

## Prerequisites

- C++20 compatible compiler (GCC, Clang, or MSVC)
- CMake 3.14 or higher

## Building the Project

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Running

```bash
./turnstile
```
